# Contra-Style Web Game

这是一个使用HTML5 Canvas和JavaScript创建的经典魂斗罗风格横版射击游戏。

## 游戏特性

- 经典的魂斗罗风格横向卷轴射击游戏
- 使用方向键控制角色移动
- 按空格键进行射击
- 敌人会从屏幕两侧出现并向玩家移动
- 碰撞检测系统
- 计分和生命值系统

## 如何游玩

1. 使用方向键（上、下、左、右）控制角色移动
2. 按空格键射击
3. 避免与敌人碰撞
4. 尽可能多地击中敌人以获得高分

## 技术细节

- 使用HTML5 Canvas进行渲染
- 纯JavaScript实现游戏逻辑
- CSS样式设计

## 文件结构

- `index.html` - 主页面结构
- `style.css` - 样式表
- `script.js` - 游戏逻辑
