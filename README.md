# Contra-Style Web Game / 魂斗罗风格网页游戏

This is a classic Contra-style side-scrolling shooter game created using HTML5 Canvas and JavaScript by Clawd AI Assistant.

这是一个由AI助手Clawd使用HTML5 Canvas和JavaScript创建的经典魂斗罗风格横版射击游戏。

## Game Features / 游戏特点

- Classic Contra-style side-scrolling shooter gameplay
- Arrow keys to control the player character
- Spacebar to shoot
- Enemies approach from both sides of the screen
- Collision detection system
- Scoring and lives system

- 经典的魂斗罗风格横向卷轴射击游戏玩法
- 方向键控制玩家角色
- 空格键射击
- 敌人从屏幕两侧接近
- 碰撞检测系统
- 计分和生命值系统

## How to Play / 游戏方法

1. Use arrow keys (up, down, left, right) to move the character
2. Press spacebar to shoot
3. Avoid colliding with enemies
4. Shoot as many enemies as possible to earn points

1. 使用方向键（上、下、左、右）移动角色
2. 按空格键射击
3. 避免与敌人碰撞
4. 尽可能多地射击敌人以获得分数

## Technical Details / 技术细节

- Rendered using HTML5 Canvas / 使用HTML5 Canvas渲染
- Pure JavaScript for game logic / 纯JavaScript实现游戏逻辑
- CSS for styling / CSS进行样式设计
